# partially_labelled_WMH_ISL_seg
Code for the paper "Comparative evaluation of training strategies using partially labelled datasets for segmentation of white matter hyperintensities and stroke lesions in FLAIR MRI". Model checkpoints and inference code can be found at:

https://github.com/Jesse-Phitidis/WMH_ISL_SEG

Please cite:

Phitidis, J., Smithard, A. Q., Whiteley, W. N., Wardlaw, J. M., Bernabeu, M. O., & Valdés Hernández, M. (2026). Comparative evaluation of training strategies using partially labelled datasets for segmentation of white matter hyperintensities and stroke lesions in FLAIR MRI. arXiv. https://arxiv.org/abs/2601.20503
